a note on the methodology

"best-selling" microchips don't get a billboard chart. semiconductor unit volumes are private, fragmented across foundries, and most chips are sold in tiny quantities to industrial buyers. so we picked the chip each year that historians, engineers, and makers would reach for as the year's defining part. early years lean industrial and computing... the modern era leans toward maker silicon, because that's the lineage adafruit cares about most. for the contested years (you can argue all of them) the source link explains the call.

play with these chip lineages today

a lot of these chips are dead silicon you can only find on ebay or in a museum. some are still in active production fifty-plus years later. and a few are the working heart of every adafruit board on a workbench right now. here's where the maker-side chips on this timeline are still alive... no checkout flow, just the lineage. each line links to the relevant adafruit learn guide so you can read first, buy second.

ATmega328P · 2008–present

Adafruit Metro 328

the arduino uno's chip in adafruit's metro form factor. still the cleanest way to learn embedded c on the chip that lit the maker movement.

learn guide → adafruit metro overview
SAMD51 · ARM Cortex-M4F

Metro M4 Express

the post-AVR step up. 120 MHz cortex-M4F, hardware float, runs circuitpython natively. this is the chip that bridged 8-bit nostalgia and 32-bit reality for a lot of makers.

learn guide → metro m4 express
RP2040 · 2021

Feather RP2040 / KB2040 / QT Py RP2040

raspberry pi's first microcontroller. dual cortex-M0+ at 133 MHz, eight programmable I/O state machines that act like tiny FPGAs. adafruit ships it in feather, kee-boar, and qt-py form factors so you can use the same chip from breadboard to keyboard build.

learn guide → feather rp2040
RP2350 · 2024

Metro RP2350 / Feather RP2350

the second-gen pi silicon. dual ARM cortex-M33 plus dual hazard3 RISC-V cores you can hot-swap between, hardware secure boot, more PIO state machines. the first microcontroller most makers will own that has both ARM and RISC-V on the same die.

learn guide → metro rp2350
ESP32-S3 · 2021

Feather ESP32-S3 / QT Py ESP32-S3

espressif's AIoT chip. dual xtensa LX7 at 240 MHz with vector instructions for on-device ML, wi-fi plus bluetooth 5 LE, native USB. if your project needs to talk to the internet and run a small neural net, this is the chip.

learn guide → feather esp32-s3
nRF52840 · 2018

Feather nRF52840 Sense

nordic's bluetooth low energy workhorse. ARM cortex-M4F, BLE 5, thread, zigbee, on-chip USB. the sense variant adds a stack of motion, light, and environmental sensors. matter-over-thread maker projects live here.

learn guide → feather sense
